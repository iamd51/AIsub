## 🎯 主要更新

### 🔧 硬編碼路徑修復 (重大改進)
解決了使用者回報的路徑錯誤問題，現在程式可以在任何目錄下正常運行。

#### 修復後的改善：

## 🚀 使用者體驗改善

### 安裝流程
```bash
# 1. 下載專案
git clone https://github.com/your-repo/AIsub.git
cd AIsub

# 2. 一鍵安裝
雙擊 install_whisper.bat

# 3. 檢查安裝
雙擊 check_installation.bat

# 4. 啟動程式
雙擊 start_whisper_gui.bat
```
